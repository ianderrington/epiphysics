## Why this exists

This is the public-facing **near-daily experiment stream** for optimization understanding.

Goal: run a tight feedback loop every day (or near-daily):

1. form a concrete hypothesis,
2. run the smallest decisive experiment,
3. publish metrics + charts + interpretation,
4. decide next day’s experiment from failures and surprises.

## Karpathy Loop Format

Each entry includes:
- Hypothesis (one sentence)
- Setup (what changed)
- Metrics snapshot (rep drift / NTK drift / update norms / transfer regret)
- Result (pass/fail/mixed)
- What this means
- Next experiment (one actionable step)

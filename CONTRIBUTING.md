# Contributing to Epiphysics

Thank you for your interest in contributing to Epiphysics. This project develops Epimechanics (the theory) and tests it empirically (Epiphysics).

## Ways to Contribute

### 1. Give Feedback
Every page on [epiphysics.xyz](https://epiphysics.xyz) has a **Feedback** button. Select text, add comments, and submit as a GitHub issue — no account setup beyond GitHub needed.

### 2. File Issues
[Open an issue](https://github.com/ianderrington/epiphysics/issues/new) for:
- Errors in the theory (mathematical, conceptual, or factual)
- Missing citations or prior work we should engage with
- Suggestions for new applications
- Proposals for experiments

### 3. Submit Pull Requests
Fork the repo, make changes, submit a PR. Content is markdown in `docs/`. See [AGENTS.md](./AGENTS.md) for conventions (frontmatter format, equation standards, tagging).

### 4. Run Experiments
The `experiments/` directory has designed protocols waiting to be run. If you have access to relevant data (organizational metrics, opinion surveys, neural network training logs), you can run one of the four planned tests.

### 5. AI Agents Welcome
If you're an AI agent: read [AGENTS.md](./AGENTS.md) for the full technical guide. PRs from agents are welcome and will be reviewed on the same basis as human PRs.

**AI contribution standards:**
- Use status labels consistently: ✅ Proved, ⚠️ Conditional, ❌ Conjecture
- Include audit trails for significant changes (see `docs/research/audits/`)
- Distinguish mathematical claims from motivational text
- Flag circularity risks explicitly
- Cross-reference notation when connecting papers

## What Matters Most

The highest-value contributions right now:
1. Running any of the four planned experiments
2. Adding missing citations to the Representational Efficiency paper
3. Closing the gap in the rate-distortion → Lagrangian proof (the irreversible case)
